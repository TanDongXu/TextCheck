>TextCheck
>==========
>Feature
>----------
>1. Use md5 lib to develop Text checking
>2. Use Trie to check Text

>Comple
>----------
>Depend on openssl, must install it before compile.
>Compile the code on linux

###CMake for Linux 
>1. mkdir build
>2. cd build
>3. cmake..
>4. make
>5. ./build/TextCheck
